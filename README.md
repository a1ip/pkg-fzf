# fzf

A command-line fuzzy finder written in Go - Plugin for [Oh My Fish][omf-link].

## Install

On OSX using `homebrew`:

```fish
$ brew install fzf
```

```fish
$ omf install fzf
```

## Usage

```fish
$ fcd
$ fh
$ fls
```
